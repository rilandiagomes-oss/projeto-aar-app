# 🩺 Projeto AAR App
**Software de Apoio à Decisão na Profilaxia da Raiva Humana**

## 📌 Descrição
O Projeto AAR App é uma ferramenta desenvolvida para auxiliar profissionais de saúde na tomada de decisão durante o atendimento antirrábico humano. O sistema fornece orientações baseadas em protocolos do Ministério da Saúde, contribuindo para a padronização das condutas e a segurança do paciente.

## 🎯 Objetivo
Apoiar a indicação correta da profilaxia da raiva humana, promovendo maior eficiência, precisão e qualidade na assistência em saúde.

## 🛠️ Tecnologias Utilizadas
- **HTML** – Estrutura da aplicação
- **CSS** – Estilização da interface
- **JavaScript** – Lógica de decisão e interatividade

## 🌐 Acesso ao Sistema
Em breve disponível online via GitHub Pages.

## 📂 Estrutura do Projeto
- `index.html` – Página principal
- `animais-transmissores-raiva.html` – Informações sobre animais transmissores
- `assets/` – Arquivos de estilo, scripts e imagens
